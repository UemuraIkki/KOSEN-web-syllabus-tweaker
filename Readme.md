# 高専Webシラバスフィルタ (Kosen Web Syllabus Tweaker)

高専のWebシラバスシステム（`syllabus.kosen-k.go.jp`）において、科目一覧を指定した条件（学年・開講期・単位数）で動的に絞り込み表示する Chrome 拡張機能です。  
目的の科目を素早く見つけ出すためのサポートをします。

---

## 機能 (Features)

シラバスの科目一覧テーブルの上部に専用の絞り込みパネルを追加し、以下の条件で **AND 検索**（該当しない行の非表示化）を行います。

- **学年**: 1年 〜 5年
- **開講期**: 前期 / 後期
- **単位数**: 1単位 〜 4単位

絞り込み条件は、すべての `<tr>` を対象にリアルタイムで適用されます（対象テーブルは `id="sytablenc"`）。  
既存の Web ページのレイアウト（DOM 構造）を書き換えず、**テーブル直前に独立したパネルを挿入する安全な設計**になっています。

---

## 対応環境 (Requirements)

- **ブラウザ**: Google Chrome（または Manifest V3 対応の Chromium 系ブラウザ）
- **対象サイト**: `https://syllabus.kosen-k.go.jp/*`
- **Manifest Version**: v3

現在、この拡張機能は Chrome ウェブストアには公開されていません。  
以下の手順でブラウザに直接読み込ませて（ローカルインストールして）使用してください。

### 1. ファイルのダウンロード

このリポジトリの右上にある緑色の **「Code」** ボタンをクリックし、**「Download ZIP」** を選択してファイルをダウンロードします。  
ダウンロードした ZIP ファイルは、任意の場所（ドキュメントやデスクトップなど）に解凍（展開）してください。

```bash

### 2. Chrome の拡張機能管理ページを開く
Google Chrome を開き、URL バーに以下を入力して Enter キーを押します。
\`chrome://extensions/\`

## 3. デベロッパーモードをオンにする
画面右上にある **「デベロッパーモード (Developer mode)」** のスイッチをクリックして「オン」にします。

## 4. 拡張機能を読み込む
デベロッパーモードをオンにすると、画面左上に 3 つのボタンが表示されます。
その中の **「パッケージ化されていない拡張機能を読み込む (Load unpacked)」** をクリックします。
ファイル選択ダイアログが開くので、ステップ 1 で解凍したフォルダ（`manifest.json` が直接入っている階層のフォルダ）を選択してください。

以上でインストールは完了です。

## 使い方 (Usage)

1. 高専 Web シラバスの「一般科目」や「専門科目」などの科目一覧ページ（URL が `https://syllabus.kosen-k.go.jp/*` で始まるページ）にアクセスします。
2. 科目一覧テーブル（`id="sytablenc"`）の直前に、「学年」「開講期」「単位数」と「リセット」が並んだ専用の絞り込みパネルが自動的に表示されます。
3. 各プルダウンを変更すると、その条件にマッチしない行（科目）が **リアルタイムで非表示** になります。
4. **「リセット」ボタン** を押すと、すべての絞り込みが解除され、全件表示に戻ります。

---

## 内部仕様 (How it works)

対象テーブルは `<table id="sytablenc">` です。
先頭の 4 行はヘッダ行として扱い、5 行目以降の `<tr>` のみをフィルタ対象とします。

各行の判定ロジックは次のとおりです。

* **単位数**: 各行の 6 番目の `<td>`（`td:nth-child(6)`）の `innerText` を使用します。
* **学年・開講期**: 各学年ごとに 4 要素の `<td>` 群を持つクラスを使用します。
    * 1年: `td.c1m`
    * 2年: `td.c2m`
    * 3年: `td.c3m`
    * 4年: `td.c4m`
    * 5年: `td.c5m`

それぞれの 4 要素に対して以下の判定を行います。
* インデックス 0（1 番目）の `innerText` が空でなければ、その科目は「その学年の前期」に開講。
* インデックス 2（3 番目）の `innerText` が空でなければ、その科目は「その学年の後期」に開講。

「学年」「開講期」「単位数」の 3 条件は **AND 検索** で評価され、条件に一致しない行は `display: none` によって非表示になります。
既存のパネルヘッダなどのレイアウトは一切変更せず、テーブル直前にコントロールパネル（`<div>`）を追加するだけの安全な DOM 操作になっています。

## 開発メモ (Development)

### `manifest.json`
* Manifest V3 準拠。
* `https://syllabus.kosen-k.go.jp/*` に対して `content.js` を `document_idle` タイミングで実行します。

### `content.js`
* ページ内の `<table id="sytablenc">` を検出し、その直前に絞り込みパネルを挿入します。
* パネルは次のスタイルで描画されます。
    ```css
    display: flex; gap: 15px; padding: 10px 15px; margin-bottom: 15px; 
    background-color: #f9f9f9; border: 1px solid #ddd; border-radius: 4px; align-items: center;
    ```
* エラー発生時でも極力ページの描画を阻害しないよう、フィルタリング処理は `try ... catch` で保護しています。
